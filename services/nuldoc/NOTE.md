# My note

## Commands

Generate the site.

```
$ ./nuldoc build
```

Create a new post.

```
$ ./nuldoc new post
```

Create a new slide.

```
$ ./nuldoc new slide
```

Update PDF.js.

```
$ curl -o static/pdf.min.mjs https://unpkg.com/pdfjs-dist@5.4.449/build/pdf.min.mjs
$ curl -o static/pdf.worker.min.mjs https://unpkg.com/pdfjs-dist@5.4.449/build/pdf.worker.min.mjs
```
